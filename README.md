# Exams

- Instalar vscode java
- Instalar https://sdkman.io/
- Instalar java 16 con sdkman: `sdk install java 16.0.1.hs-adpt`
- Crear BD, conectar a psql y luego: `create database exams;`
- Levantar app: `./mvnw spring-boot:run`
